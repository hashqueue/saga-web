# saga-web

一站式项目管理平台 - developed with `Vue3`、`Ant Design Vue 4.x` and `Vite4`.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
